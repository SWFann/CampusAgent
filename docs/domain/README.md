## 📝 词汇表说明

词汇表是项目的唯一权威术语来源，所有代码、API、文档和讨论应使用词汇表定义的术语。

### 如何使用

1. **编写代码时**：变量名、函数名使用英文标识符
2. **编写文档时**：首次出现时提供中英文对照
3. **讨论时**：使用标准术语避免歧义
4. **评审时**：检查是否符合词汇表定义

### 更新流程

1. 发现术语不一致或需要新增术语
2. 在词汇表中添加或修正
3. 提交 PR 并附上变更说明
4. 团队评审通过后更新版本号
5. 同步更新相关文档和代码注释
