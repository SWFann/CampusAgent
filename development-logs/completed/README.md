# 已完成任务记录

本目录是过程证据归档，不是当前项目状态的权威来源。当前阶段状态请看 `docs/project/`。

## 子目录

| 子目录 | 内容 |
|---|---|
| `p0-foundation/` | P0 原始基础文档任务记录 |
| `p1-engineering/` | P1 工程初始化任务记录 |
| `remediation-r0/` | 整改基线和安全检查记录 |
| `remediation-r1/` | P0 契约、角色、API、威胁模型整改记录 |
| `remediation-r2/` | 后端、测试和 Python 依赖整改记录 |
| `final-review-r4/` | 最终验收或复审记录 |
| `codex/` | Codex 独立整改记录 |

## 查找方式

优先按任务编号查找，例如 `R1-06`、`P1-04`。如果一个任务既有 `in-progress` 又有 `completed` 记录，以 `completed` 中最新记录为历史证据，但仍需回到 `docs/project/` 判断当前状态。
